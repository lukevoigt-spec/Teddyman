# Background prompts — optimized for ChatGPT Plus (4:3, one-paste each)

Each block below is **fully self-contained** — the style preamble is baked into
every one, so you just **paste ONE block, generate, download, rename, upload.**
No extra lines, no juggling. They all share an identical style spec, so the whole
game (both acts) comes out looking like one cohesive set.

- **Aspect ratio:** 4:3 landscape (already in every prompt).
- **Format:** download as `.png` (works as-is — no conversion).
- **Rename** each to the filename shown, then drop it in this `art/` folder.
- If an image shows text or a person → reply **"Regenerate — zero text/letters,
  no people, environment only."**
- *(Optional, for max consistency: generate them all in ONE ChatGPT conversation.
  Not required — the baked-in style keeps them matching either way.)*

Priority: `bg-title` → `bg-learn` → `bg-base` → `bg-city` → `bg-battle` →
`bg-victory` → `bg-rest` → `bg-intro`, then the Act-2 `-a2` set, then `icon`.
