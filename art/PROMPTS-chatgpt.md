# Background prompts — optimized for ChatGPT Plus (DALL·E / GPT-4o images)

Paste **one block at a time** into ChatGPT. Each is self-contained. Download the
result, **rename to the filename shown**, and drop it in this `art/` folder
(`.png` is fine — no conversion needed).

### Best workflow (gives a cohesive set)
1. Do **`bg-title`** first. When you like the style, keep going **in the same
   ChatGPT conversation**.
2. For every later one, start the message with:
   **"Same exact art style, palette, line quality, and rendering as the previous
   image. Now create:"** — then paste the block. This keeps all scenes matching.
3. If any image shows text or a person, reply: **"Regenerate — zero text/letters
   and no people; environment only."**
4. Size: **landscape 16:9** (the game crops to fit). Keep the center open.

Priority order: `bg-title` → `bg-learn` → `bg-base` → `bg-city` → `bg-battle` →
`bg-victory` → `bg-rest` → `bg-intro`, then the Act-2 `-a2` set.
